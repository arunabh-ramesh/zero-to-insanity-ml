# zero-to-insanity-ml
A curated collection of my machine learning projects, implementations, and key learnings. please hire me.

The purpose of this github repository is to showcase my progression through learning ml from zero to "insanity" (This is the coolest name I could think of).

My goal is to get an overall understanding of the underlying concepts of machine learning, and be able to apply this learnt knowledge by completing projects.

The rules I will set upon myself for this project include:
- Learning* one new thing every week (I can learn more than one thing, but the lowest is one).
- The use of LLM's (ChatGPT, Claude, Gemini, etc.) should minimized to 0 if commpletely possible. 
- If I do use an LLM, I will be required to document what I used it for and why I decided to use it.
- Any resource that I use to learn must be included in this README.

*Learning - My definition of learning is being able to apply the skill within another project without needing to refer back to the original learning material. (This definition may change)
